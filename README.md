# DriveBy-Multiplayer-Game
DriveBy - 2d Multiplayer Game / Node.js &amp; socket.io <br />
<br />
Drive: WSAD or arrows <br />
Shoot: Space <br />
R: Reload <br />
T: Chat <br />
Hold Q: Shoot left <br />
Hold E: Shoot right <br />
Z or 1-7: Change weapons <br />
Esc: Menu <br />
Mouse click your car to change it
